## API Reference

#### Get all items

```http
  POST /api/v1/user
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `sign-Up` | `string` | **Required**. Your API key |

#### Get item

```http
  GET /api/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
